In Which Various Automated Tools Fail In Interesting Ways

COPYA

immediate disk read error

Locksmith Fast Disk Backup

unable to read any track

EDD 4 bit copy (no sync, no count)

works

Copy ][+ nibble editor

modified address and data epilogues ("DA AA EB" instead of "DE AA EB")

Disk Fixer

["O" -> "Input/Output Control"] set Address Epilogue to "DA AA EB" set Data Epilogue to "DA AA EB" all tracks readable T00 -> looks like a DOS 3.3 RWTS T11 -> DOS 3.3 disk catalog T01,S09 -> startup program is "EDU-WARE"

Why didn't COPYA work?

modified epilogue bytes (every track)

Why didn't Locksmith FDB work?

modified epilogue bytes (every track)

EDD worked. What does that tell us?

no half or quarter tracks almost certainly no nibble check (just structural changes to epilogue)
